## 遇到的问题和解决方案

### 问题 1：ECharts 类型定义错误（2026-03-10 下午）

**错误信息**：
```
TS2339: Property 'ECharts' does not exist on type 'typeof import("echarts")'
```

**原因**：
ECharts v5 不再将 `ECharts` 作为 class 导出，不能用 `InstanceType<typeof echarts['ECharts']>`。

**解决方案**：
改用 `ReturnType<typeof import('echarts')['init']>` 推断实例类型。

```typescript
// 错误写法
let chart: InstanceType<typeof import('echarts')['ECharts']> | null = null

// 正确写法
let chart: ReturnType<typeof import('echarts')['init']> | null = null
```

**涉及文件**：`AttendanceStats.vue`、`BusinessStats.vue`

---

### 问题 2：测试数据缺失（2026-03-10 下午）

**现象**：
初始 `SEED_DATA_SQL` 仅有 2 个用户账号 + 1 条预警阈值配置。安装后是空数据库，
所有功能无法正常测试（签到列表空、统计图空白、预警列表空）。

**解决方案**：
在 `schema.ts` 中新增 `TEST_SEED_SQL`，`database.ts` 的 `seedData()` 判断全新安装后一并写入：

- 15名学生（英语A班/数学B班/科学C班，覆盖红/黄/绿三种课时状态）
- 24条购课/续费记录（含金额，用于经营报表验证）
- 150条签到记录（10次课 Mon/Wed/Fri，到课/迟到/请假/旷课均有分布）
- 6条跟进记录（pending/contacted 两种状态）

**触发条件**：`seedData()` 检测 users 表 count=0 才写入，不影响已有数据。

---

### 问题 3：attendances 表缺少唯一约束（2026-03-10 下午）

**现象**：
签到表没有 `UNIQUE(student_id, date)` 约束，可能对同一学生同一天重复插入，导致统计数据翻倍。

**解决方案**：
在 `CREATE TABLE attendances` 中增加 `UNIQUE(student_id, date)` 表级约束，
并移除了原来多余的 `idx_attendances_student_date` 普通索引（UNIQUE 约束自动建立等效索引）。

---

### 问题 4：Agent 并发开发的文件同步问题（2026-03-10 上午）

**现象**：
多 Agent 并发开发时，backend-dev 已完成 `AttendanceToday.vue`（322行），
但 team lead 在检查时读到了临时状态，误判为未完成（只有9行占位符）。

**解决方案**：
不依赖 Agent 状态汇报，直接用 `wc -l` / `head` 命令验证文件实际行数和内容作为完成判据。

---

### 问题 5：TypeScript 编译 25 处错误（2026-03-10 上午）

**错误类别及修复**：

| 错误类型 | 典型位置 | 修复方法 |
|---|---|---|
| null 安全 | repositories `rows[0]` | 改为 `rows[0] ?? null` |
| undefined 检查 | `attendance.ts` | 添加 `if (!value) return` 守卫 |
| 未使用变量 | `BottomNav.vue` | 删除未引用的 import |
| ECharts 类型 | `chartOptions.ts` | 使用 `ECBasicOption` 替代 |

**工具**：`vue-tsc --noEmit` 编译检查，`npm run build` 最终验证。

## 数据库设计关键点

### 索引（共6个，去除冗余后）

| 索引名 | 覆盖字段 | 用途 |
|---|---|---|
| `UNIQUE(student_id, date)` | 签到表 | 防重复 + 查询加速（替代普通复合索引）|
| `idx_attendances_student_id` | 签到表 | 查某学生历史 |
| `idx_attendances_date` | 签到表 | 按日期查签到 |
| `idx_purchases_student_id` | 续费表 | 查学生购课历史 |
| `idx_purchases_date` | 续费表 | 月度收入统计 |
| `idx_followup_student_id` | 跟进表 | 查学生跟进记录 |
| `idx_students_status` | 学生表 | 过滤在读/归档 |

### 课时扣减逻辑

- 到课（present）→ 扣 1 课时
- 迟到（late）→ 扣 1 课时
- 请假（leave）→ 不扣
- 旷课（absent）→ 不扣

### 生命周期标签

| 标签 | 条件 | UI 颜色 |
|---|---|---|
| `new` | 注册 ≤ 30 天 | 蓝色 |
| `active` | 课时 > 阈值 | 绿色 |
| `warning` | 0 < 课时 ≤ 阈值（默认3） | 黄色 |
| `at_risk` | 课时 = 0 | 红色 |

## 性能与加载优化

| 模块 | 方式 | 效果 |
|---|---|---|
| 所有页面 | 路由 `() => import(...)` | 首屏只加载当前页 |
| ECharts（1127KB） | `onMounted` 内 `await import('echarts')` | 进入统计页才加载 |
| xlsx（282KB） | 静态 import，但页面本身懒加载 | 进入导出页才触发 |
| Lucide 图标 | 各页面单独 named import | 只打包用到的图标 |
| 核心 bundle | Vite manualChunks 分包 | index chunk 约 165KB |
